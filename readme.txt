# DE:

Viele Gastronomen wenden täglich sehr viel Zeit auf, um neue Tagesgerichte zu verfassen und zu Papier zu bringen. 
Viele aktualisieren aber nicht die Webseite weil es Ihnen einfach zu umständlich oder zu kompliziert ist.

Verknüpfen Sie nun, Ihre WordPress Seite mit Ihrer GASTROFIX Kasse.

Präsentieren Sie ausgewählte Produkte aus der GASTROFIX Kasse direkt auf Ihrer Internet Präsenz.
Editieren Sie Ihre GASTROFIX Stammdaten und halten Sie die Produktenamen, Beschreibungen und Preise, durch einen Klick aktuell.

 

## Funktionsweise
- Das GASTROFIX-Wordpress-Plugin steht allen Marketingunternehmen/Webdesignern die mit WordPress arbeiten zur Verfügung.
- Das GASTROFIX-Wordpress-Plugin holt via GASTROFIX-API die gesamten Artikeldaten ab.
- Es können im Plugin Artikel per Filtertechnik einzeln oder ganze Artikel-Gruppen deaktiviert werden (somit ist es möglich Infotexte nicht auf der digitalen Speisekarte anzuzeigen, wie auch bspw. Saisonale Artikel etc.).
- Auch Allergene und Nährwerte werden aus der GASTROFIX Artikelinfo mit übertragen und auf Ihrer Internetseite angezeigt.
- Fügen Sie zusätzlich beim Text der Allergene auch eine Weinempfehlung dazu, inkl. Weinbeschreibung und Bild der Weinflasche. 
- Eine Standart-Formatierung ist bereits vorbereitet, womit der Webdesigner bereits arbeiten kann, er kann diese jederzeit im WordPress Editor optimieren.

## How it works
- The GASTROFIX WordPress plugin is available to all marketing companies/web designers working with WordPress.
- The GASTROFIX WordPress Plugin collects all article data via the GASTROFIX API.
- In the plugin, articles can be deactivated individually or entire article groups via filter technology (thus it is possible not to display info texts on the digital menu, such as seasonal articles, etc.).
- Allergens and nutritional values are also transferred from the GASTROFIX article info and displayed on your website.
- Add a wine recommendation to the text of the allergens, incl. wine description and picture of the wine bottle. 
- A standard formatting is already prepared, with which the web designer can already work, he can optimize this at any time in the WordPress Editor.
